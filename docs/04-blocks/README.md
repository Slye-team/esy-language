There are many blocks we want to make ([see this](https://github.com/Slye-team/esy-language/blob/master/README.md)), but 
till now we have these blocks:

1. [Timers](./01-timers.md) (Timeout/Interval)
2. Env
3. Enc
4. Wait
5. Promise
